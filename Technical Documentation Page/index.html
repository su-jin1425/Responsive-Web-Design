Introduction

CSS are stylesheets used to give styles to websites.

CSS means Cascading Stylesheets.

Some CSS attributes are:

  • font-family
  • font-size
  • color
  • width
  • padding
Selectors

To style HTML elements in CSS, selectors are key.

There are different types of selectors such as class selectors, id selectors, and element selectors.

- Class selector syntax:

.className { ... }

- Id selector syntax:

#idName { ... }

- Element selector syntax:

elementName { ... }
Flexbox

Flexbox is a layout module that allows for the efficient distribution of space within a container.

Some Flexbox properties are:

justify-content: It takes values like:

flex-start, flex-end

align-items: It takes values like:

flex-start, center, flex-end

flex-direction: It takes values like:

column, row, column-reverse, row-reverse
Colors

In CSS, you can set the color of elements and text.

There are three common methods to specify colors:

  • Named colors
  • RGB
  • Hexadecimal
Text Styles

To style text, you can use the following properties:

  • font-family
  • font-size
  • font-weight
  • line-height
  • color
Media Queries

Media queries are used to make websites responsive.

- The media query syntax:

@media (max-width: 600px) { ... }
